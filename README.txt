== Description ==

Socialo Cricket Live Score is a powerful WordPress plugin that seamlessly integrates real-time cricket match scores into your website, powered by the SportMonks API. Engage your audience with live updates and detailed team information through an elegant user interface.

== Installation ==

1. Upload `socialo-cricket-live-score.php` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Place `<?php do_action('socialo_cricket_live_score_hook'); ?>` in your templates.

== Frequently Asked Questions ==

= How do I display live cricket scores on my posts or pages? =

To display live cricket scores, use the following shortcode:

```shortcode
[socialo_cricket_live_score]
```

= How do I add the live cricket score widget to my sidebar? =

1. Navigate to WordPress Admin > Appearance > Widgets.
2. Add the "Socialo Cricket Live Score" widget to any available widget area.
3. Customize the widget options and save.
